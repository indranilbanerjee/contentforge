#### 2.2 Inject Sentence Variety (Burstiness)

**Human writing has "bursts" — short sentences followed by long, then medium, etc.**

**Example of TOO UNIFORM (robotic):**
```
❌ Multi-agent AI systems deploy specialized agents for distinct sub-tasks. (10 words)
This architecture enables optimization for specific functions. (8 words)
Each agent focuses on one aspect of the content pipeline. (11 words)
The system coordinates these agents to produce final output. (10 words)

→ All sentences 8-11 words, feels mechanical
```

**Humanized version (burstiness):**
```
✅ Multi-agent AI systems deploy specialized agents for distinct sub-tasks. (10 words)
This matters. (2 words) [SHORT PUNCH]
Unlike single-model approaches that force one AI to handle everything—research, drafting,
fact-checking, editing—multi-agent systems let each component specialize. (22 words) [LONG FLOW]
The result? Higher quality with less compromise. (7 words) [SHORT IMPACT]

→ 10 words → 2 → 22 → 7 = Natural human rhythm
```

**Technique: Break up medium sentences into short + long combos**

**Before (uniform):**
```
❌ "The McKinsey study analyzed 200 agencies and found quality scores averaging 7.5 out of 10. (15 words)
This represents a significant improvement over single-model systems which scored 6.2 out of 10. (15 words)
The performance gap stems from specialization advantages inherent in multi-agent architectures. (12 words)"

→ All 12-15 words, monotonous
```

**After (bursty):**
```
✅ "The McKinsey study analyzed 200 agencies. Quality scores? 7.5 out of 10 for multi-agent systems. (16 total, but broken: 8 + 8)
That's a significant jump from the 6.2 that single-model systems achieved. (13 words)
The performance gap stems from specialization—when each agent optimizes for its specific function rather than compromising across competing objectives, output quality rises. (25 words)

→ Pattern: SHORT → MEDIUM → LONG = natural rhythm
```

#### 2.3 Calculate Burstiness Score

**Burstiness formula (simplified):**
```
Burstiness = Standard Deviation of Sentence Lengths / Mean Sentence Length
```

**Target:** ≥0.7 (higher = more human-like variation)

**Example calculation:**
```
Sentences: [8, 22, 7, 18, 25, 5, 20, 14, 28, 6, ...]
Mean: 15.3 words
Standard Deviation: 8.1
Burstiness: 8.1 / 15.3 = 0.53

→ ⚠️ Below 0.7 target, too uniform
```

**To improve: Add more short sentences (3-7 words) to increase variation**

### Step 4: Vary Sentence Openings

**AI writing often uses repetitive sentence starts. Fix this.**

#### 4.1 Detect Repetitive Patterns

**Scan for consecutive sentences starting the same way:**

**Problem patterns:**
```
❌ "The system uses specialized agents. The architecture enables optimization. The results show quality improvements."
→ Three consecutive "The [noun]" openings

❌ "Multi-agent systems offer benefits. Multi-agent systems reduce costs. Multi-agent systems improve quality."
→ Three consecutive same subject openings

❌ "Additionally, costs decrease. Additionally, quality improves. Additionally, scale increases."
→ Three consecutive same transition word
```

#### 4.2 Diversify Sentence Starts

**10 sentence opening patterns to rotate:**

1. **Subject-Verb:** "Agencies implement AI..."
2. **Prepositional Phrase:** "In 2026, adoption accelerated..."
3. **Adverb:** "Significantly, quality scores rose..."
4. **Participial Phrase:** "Building on earlier findings, researchers..."
5. **Infinitive Phrase:** "To reduce costs, agencies..."
6. **Dependent Clause:** "When agencies implement AI, they see..."
7. **Question:** "Why does this matter?"
8. **Transition:** "However, challenges remain..."
9. **Number/Data:** "73% of agencies now use AI..."
10. **Direct Object:** "Cost reduction drives adoption..."

**Example transformation:**
```
❌ REPETITIVE:
"Multi-agent systems use specialized agents.
Multi-agent systems achieve higher quality.
Multi-agent systems reduce operational costs."

✅ VARIED:
"Multi-agent systems use specialized agents. [Subject-Verb]
Quality? It jumps to 7.5 out of 10. [Question + Data]
By specializing each component, costs drop 68%. [Prepositional Phrase + Data]"
```

### Step 5: Add Human Conversational Elements

**Make content feel like a knowledgeable human wrote it, not an AI.**

#### 5.1 Strategic Questions

**Add rhetorical questions to engage readers:**

**If brand voice allows second-person (check brand profile):**

```

✅ ENGAGING: "Your content team probably faces mounting pressure—more requests, same budget, no time."
```

**⚠️ Only use if brand profile specifies second-person ("you") voice**

#### 5.3 Parenthetical Asides

**Add brief clarifications or emphasis:**

```
✅ "Quality scores (measured on a 10-point scale) jumped from 6.2 to 7.5."

✅ "The 68% cost reduction—not a typo—stems from automation and specialization."

✅ "Implementation takes 6-8 weeks (faster than most teams expect)."
```

**Placement:** 1-2 per article, sparingly

#### 5.4 Sentence Fragments for Emphasis

**Human writers use intentional fragments for punch:**

```
✅ "Multi-agent systems deliver three key benefits. Cost reduction. Quality improvement. Scalability."

✅ "The choice is clear. Adapt or fall behind."

✅ "What about quality? Not just maintained. Improved."
```

**⚠️ Use sparingly (2-3 per article max) and only for dramatic emphasis**
